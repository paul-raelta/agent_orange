Agent Orange Help

Autonomous earnings intelligence — your investing desk, on autopilot. This guide walks through every feature of the tool, with annotated screenshots pointing out each important component and a short “how to use it” for each.

What Agent Orange does

You keep a watchlist of companies. Agent Orange assigns an AI agent to each one, and that agent runs the same five-step loop every reporting season — so you get validated numbers, with receipts, the moment results drop.

STEP 1
Locate
Find where this company’s results live — its IR site + SEC EDGAR.
STEP 2
Monitor
Poll on a cadence that intensifies inside the predicted filing window.
STEP 3
Extract
Pull the figures that move trades: revenue, net income, EPS, margins.
STEP 4
Validate
Cross-reference each figure; agreement earns confidence.
STEP 5
Surface
Show it with full provenance; route conflicts to you.

That’s the whole tool. Every number you see links back to the document it came from — that’s the point of Agent Orange.
